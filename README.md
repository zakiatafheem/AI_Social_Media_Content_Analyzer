# 🚀 AI Social Media Content Analyzer

## 📌 Project Overview
The AI Social Media Content Analyzer is a Generative AI-powered web application that analyzes social media content using Large Language Models (LLMs).

The system can automatically generate:
- Sentiment Analysis
- Topic Detection
- Content Type Classification
- AI-generated Summaries
- Engagement Suggestions
- Hashtag Recommendations
- Engagement Score

The project uses the Groq API with the Llama 3.1 model to provide fast and intelligent social media content analysis in real time.

The application is built using Python and Streamlit and deployed on Hugging Face Spaces.

---

# 🎯 Problem Statement
Social media platforms generate huge amounts of content every day. Businesses and creators often struggle to quickly understand audience sentiment, engagement quality, and content effectiveness.

This project aims to solve that problem by building an AI-powered analyzer that can:
- Understand social media text
- Detect sentiment and topics
- Generate summaries
- Suggest engagement improvements
- Recommend hashtags automatically

This helps content creators and businesses improve audience engagement and content strategy.

---

# 📂 Dataset Details
This project does not use a traditional machine learning dataset.

Instead, it uses:
- User-provided social media text input
- Prompt engineering techniques
- Large Language Model responses through the Groq API

The AI model analyzes the input dynamically in real time.

---

# 🧠 Model / Architecture Used

## Technologies Used
- Python
- Streamlit
- Groq API
- Llama 3.1 8B Instant Model
- Prompt Engineering
- Generative AI

## Application Workflow
1. User enters social media content
2. Prompt is sent to the Groq API
3. Llama 3.1 model analyzes the text
4. AI generates:
   - Sentiment
   - Topic
   - Content Type
   - Summary
   - Engagement Suggestions
   - Hashtags
   - Engagement Score
